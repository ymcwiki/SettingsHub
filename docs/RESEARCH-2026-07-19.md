# 玩家痛点调研与可用性计划(2026-07-19)

grok CLI(grok-4.5)六维调研玩家真实诉求,交叉插件知识库与现有功能,定出下一轮可用性迭代。

## 方法与可信度

- D1 隐藏 CVar、D4 预设需求:grok 联网搜 X + Reddit + 论坛,**带链接引用**(可信度高)。
- D2 迁移备份、D3 补丁漂移、D5 AIO 接班信任、D6 设置不生效:grok 联网搜索档口被限流(stopReason=Cancelled),改用 grok 训练知识作答(它训练语料重度覆盖 X 与游戏论坛,答出的 CVar 名、插件名、玩家原话都具体可查)。**未逐条带链接,但都是稳定、公认的社区共识**,当方向证据用,不当独家事实用。
- 原始输出留存 job 目录,不进交付。

## 六维发现 × SettingsHub 现状

### D1 玩家最想要的隐藏设置(搜索引用)
最想要的 8 项:最大视距倍率、SpellQueueWindow(施法队列/延迟容忍)、姓名板距离、FOV、关全屏光效/泛光、死亡去色、姓名板堆叠(overlapH/V)、报错/喊话文字开关。**核心洞察:玩家不要"全部 CVar 转储",要一份"短、有解释、有名字"的清单**——什么作用、安全范围、正版/怀旧上限、怎么重置。AIO 的短板正是"给了原始列表却零解释,你得先知道名字"。
> 现状:已有 152 条策展 + 发现页 8 条意图引导。缺一个"最想要的隐藏设置"精选入口(把这 8 项聚成一屏)。

### D2 备份/迁移/分享(训练知识)
痛点原话:"重装后 UI 全没了""换新电脑怎么搬""主号调好怎么克隆给小号""怎么导出设置"。现有做法各有摩擦:
- **拷 WTF 文件夹**:路径带账号名、改名/转服就断、部分拷贝留半坏 UI、云同步 + 开着游戏 = SV 损坏、无法"只搬键位/只搬画质"。
- **ElvUI/Myslot 式插件档案**:只覆盖自己那套;**CVar(画质/声音/跳字/镜头/姓名板)基本没人管**。
- **Discord 分享串**:巨长、版本错位、"导入失败"、没有"整客户端快照"、给队友不是"丢过去就好"。
> 核心洞察:**CVar 是迁移盲区;"我的设置"= 6~15 条导出路径,不是一条**。账号级 vs 角色级不透明,玩家猜错,每个小号重设一遍。
> 现状:已有命名 profile + 导入导出串——正好补这个盲区。**但审查发现导入有 bug(P1 #8/#9/#11:bulk 域没存进 profile、勾选框不门控、未知项照写),迁移这条核心价值链现在是漏的。**

### D3 补丁改/重置设置(训练知识)
原话:"补丁后键位全没了""画质又自己重置了""暴雪没问就改了我的姓名板""每次补丁插件都被禁用""12.0 前置补丁把我 CVar 全清了"。最常被动的:键位、画质(分辨率/抗锯齿/阴影/视距/垂直同步)、姓名板 CVar、镜头/跳字、编辑模式布局、插件启用状态。
> 核心洞察:**玩家只在"感觉不对"之后才发现,恢复全靠手动或"但愿我导出过"**。
> 现状:快照 + diff 引擎已有,但"登录自动落快照 + 本次补丁改了你哪几项"报告没做(路线图 v0.6)。D3 强验证这个方向。

### D4 预设/被选项淹没(搜索引用)
最想要的场景包(按热度):团本/M+ 性能、极限帧数/弱机、PvP 减负(信息>好看)、清晰锐利画质、新手/回归极简、晕 3D 舒适。现在的来源全不满意:YouTube 清单(10~15 分钟点菜单、每补丁漂移)、Reddit/攻略(互相打架的一次性建议)、CVar/pastebin(强但不透明、劝退非高玩)、UI 导入串(只修框体不修系统画质与舒适 CVar)、自动帧数插件(证明"一键内容预设"有需求,但只偏性能、不解释权衡)。
> 现状:已有 4 个包(舒适/团本性能/PvP 信息/新手)。缺"极限帧数/弱机"与"清晰锐利画质"两个高频包。

### D5 AIO 接班 × 信任(训练知识)
玩家说 AIO"死了"指的是**维护弃坑**(不是恶意):红色 Out of Date、面板打不开/报 nil、CVar 改名后设了没用、碰安全 UI 触发"插件导致操作失败"、大版本后坏几周、大家散去用一次性 CVar 宏。设置插件坏时玩家怕:taint/战斗锁定、把 UI 搞砖、设置丢失、来路不明的"写每个 CVar"、Midnight secret 让某些状态读不到。**愿意换新插件的条件**:补丁日速度(几小时到几天更新 + 公开 changelog)、一个可信中枢(替掉 AIO + 一堆一次性)、低 taint 面(不碰战斗写入、不 hook 安全路径)、安全栏(重置/撤销、危险 CVar 警告、Midnight 读不到时说清为什么)、可证明的持久化(profile、账号 vs 角色标注、"现在生效"vs"下次登录")、社区口碑("当前版本可用"的评论)、诚实范围(说清 Midnight 砍了什么,别让玩家觉得被糊弄)。
> 现状:安全栏(撤销/重置)、低 taint、维护纪律都已具备。缺"Midnight 诚实"提示(设置灰/不可用时说清原因)、账号 vs 角色标注。多为定位与 CF 描述,少量功能。

### D6 设置不生效/被插件接管(训练知识)
六类接管:①姓名板(Plater/Kui/ThreatPlates 接管缩放/颜色/堆叠/显示)②镜头/视距(Leatrix Plus、ElvUI 每次登录/切区重刷 CVar)③地图/小地图(Mapster、HandyNotes、ElvUI、SexyMap)④动作条/键位(Dominos、Bartender、ElvUI 自带布局库)⑤UI 缩放/字体/聊天(ElvUI/WA 重写)⑥画质/高级 CVar 被跳字(SCT/xCT)、团队框体(VuhDo/Grid2/ElvUI)遮蔽,或需重启客户端才生效的加载期 CVar。玩家原话:"我设了 CVar 没反应""每次登录最大视距又回去了""Leatrix 老拽我镜头""Mapster 把一切都覆盖了""ElvUI 覆盖游戏 UI"。现在的排查:`/dump GetCVar` 确认值 → `/reload`/重启 → 二分法禁插件 → 上 r/wow/Discord 问"哪个插件管这个" → 删那插件的 SV。
> 核心洞察:**值对了但行为不对时,把"控制平面 → 拥有它的插件/模块"映射出来,指向那个插件的 UI,而不是 Esc→选项。**
> 现状:这正是路线图 v0.6 医生的"外部接管"规则(mapFade 实案就是这个)。D6 给出了完整的可落地特征库。三方独立指向同一个功能(路线图 + 我方验收实案 + 玩家调研),信号最强。

## 迭代排期(Claude 编排,codex gpt-5.6-sol 实现,每轮 Claude 复核 + 无头断言)

- **Round 1(旗舰):外部接管医生**。读-only 特征库 + 检测 + 浏览器徽章 + 医生区。三方验证,直击"食之无味"与 mapFade 实案,守"绝不改配置"红线(纯读)。→ v0.6.0
- **Round 2:迁移可用性**。修 import P1 三条(#8/#9/#11) + 每项账号 vs 角色徽章。让"迁移"这条核心价值链真的成立。→ v0.6.1
- **Round 3:补丁漂移报告**。登录检测 build 变化自动落快照 + "本次补丁改了什么"可读报告(复用快照 diff)。→ v0.6.2
- **Round 4(内容批):** +两个场景包 + 发现页"最想要的隐藏设置"精选 + Midnight 诚实提示。→ v0.6.3

红线:全部结论只预览/试穿/撤销/只读检测,绝不静默改配置;零 taint;每轮配无头断言。
