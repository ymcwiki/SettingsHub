# Changelog

## v0.3.5(2026-07-17)

- `/sh probe` 升级为几何探针:鼠标焦点链之外,加打当前页首个勾选行与勾选框的矩形/层级/启用状态、光标在控件坐标系的位置、程序化点击回环(排查 12.0.7 实机「勾选框收不到鼠标」问题,probe 已实证点击落在主窗口而非控件)

## v0.3.4(2026-07-17)

- **修复 12.0.7 上枚举为 0**(v0.3.3 实机 diag 抓到):`C_CVar.AreCVarsLoaded` 是 12.1 API,旧客户端上不存在,此前的判断把「API 缺失」当成「CVar 未加载」永远提前返回,导致浏览器页空表、快照无法建立、进世界断言失效。现在只有 API 存在且明确报未加载时才等待
- diag 报告增加枚举链路逐层信息(API 存在性 / 原始命令数 / Cvar 型匹配数)
- 新增 `/sh probe`:打印鼠标悬停处的框架链,排查点击被遮挡类问题

## v0.3.3(2026-07-17,未公开发布,排障版)

- 新增 `/sh diag`:一条命令产出完整诊断报告(环境状态、写管线回环含错误全文、全部策展项可写性扫描记名拒绝项、会话失败清单),弹窗展示一张截图带走。配合实机验收的「勾选框点不动」排障

- 推荐包应用跳过本客户端不存在的键(预览时已计入「将跳过」),不再产生失败条目
- VERIFIED.md 第 3 条增补:三个地图类 CVar 在 12.0.7 实测 scope 与 12.1.0 census 不一致,按版本差异对待

## v0.2.0(2026-07-17)

国际化与备份。实机验收清单见 docs/ACCEPTANCE.md 的 v0.2 节。

- **enUS 本地化**:全部 UI 与消息改为英文原文 + `ns.L` 译文表(zhCN/zhTW 走中文表,其他语言显示英文);106 条策展说明逐条配英文白话版(数据模型迁移为 `text = { zh, en, keywords }`);搜索索引中英描述加关键词三路全进,任何客户端语言下中英俗名都能命中;zhCN 译文覆盖完整性由无头测试扫描保证
- **命名快照**(新「快照」页):一键把全量 1,600+ 项 CVar 现状落盘(不只本插件改过的),名字 + 时间戳 + build 号,上限 10 份超限确认淘汰最旧;任意快照与当前值或另一份快照对比,五类口径与元数据管线一致(值变/新增/删除/scope/secure);值变行勾选后选择性恢复,逐条走写管线,全程可撤销、记期望态
- **冲突检测**(日志页新冲突区):重放补写时把漂移归因到 blame 记录的来源并跨登录计数,同一项被同一外部来源在 3 个不同登录里覆盖即判冲突;每条给两个处置:停止管理(移出期望态)或保持我的值(继续每次改回,计数清零重新观察)
- 修复:中文行标签按「。/:」切第一句时,Lua 字节级字符类会截断含 0x80 字节的汉字(如「速」),改用 plain find(v0.1 潜在问题,实机未曾暴露)
- 无头测试增至 117 项断言(新增 locale 行为、zhCN 覆盖扫描、快照回环与上限淘汰、冲突计数与处置)
